Markup Structure

Coursaty template is well commented and easily to understand, each page have

  1. <head> tag -  which contains the meta tags and css files.
  2. <body> tag -  which contains entire content.
  3. <header> tag - which contains logo and navigation.
  4. <footer> tag - which contains all footer content.

Layout Structure

By default, Coursaty template is a full-width based layout. However, you can set your favourite boxed layout.

  1. For (boxed 1) layout, add "bboxed" class to body tag <body class="bboxed"> then add "boxed" class to (entire) div <div id="entire" class="boxed">
  2. For (boxed 2) layout, add "bboxed" class to body tag <body class="bboxed"> then add "boxed-2" class to (entire) div <div id="entire" class="boxed-2">

Skins

Coursaty template has 'Orange' skin by default, However you can set your favourite color skin, just place the below tag before end of </head> tag.

<link class="skin_link" rel="stylesheet" href="assets/css/skin/default.css">

replace default.css with your favourite skin

  1. blue
  2. cyan
  3. green
  4. lactic
  5. pink
  6. purple
  7. red
  8. yellow
